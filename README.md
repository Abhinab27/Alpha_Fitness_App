# Alpha Fitness

A Flutter fitness application for tracking exercises, planning routines, and calculating health metrics.

## Features
- **Exercise Library**: Searchable database of 72 exercises with step-by-step instructions.
- **Routine Planner**: Structured weekly training splits (Push, Pull, Legs).
- **Active Workout Mode**: Guided workout sessions with an integrated rest timer.
- **Fitness Calculators**: Estimators for Daily Calories, Protein intake, and Hydration targets.
- **Theme Support**: Seamless switching between Midnight (Dark) and Cream (Light) modes.
- **Local Persistence**: Saves user preferences and profile data using SharedPreferences.

## Tech Stack
- **Framework**: Flutter
- **State Management**: Provider
- **Icons**: Material Icons (Optimized for stability)
- **Typography**: Google Fonts (Outfit & Lora)
- **Storage**: SharedPreferences

## Getting Started
1. Clone the repository.
2. Ensure you have the Flutter SDK installed and configured.
3. Run `flutter pub get` in the terminal to fetch dependencies.
4. Run `flutter run -d windows` (or select your target device) to start the application.
