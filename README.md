# Alpha Fitness

A comprehensive fitness application built with Flutter, designed to help users track their health and fitness journey.

## Features
- **Firebase Authentication**: Secure login and signup.
- **Workout Planner**: Access to various workout routines.
- **BMI Calculator**: Easy health metric tracking.
- **Progress Analytics**: Visual progress tracking using charts.
- **Cloud Sync**: Integration with PostgreSQL/Node.js backend via REST APIs.
- **Dark Mode**: Support for system-wide dark and light themes.

## Tech Stack
- **Frontend**: Flutter & Dart
- **State Management**: Provider
- **Database**: PostgreSQL (via Node.js API)
- **Auth**: Firebase Authentication
- **Charts**: fl_chart

## Getting Started
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to start the application.
